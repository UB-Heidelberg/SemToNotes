Drawing Shapes

In order to draw shapes, switch a drawing canvas into mode create by calling drawing.setModeCreate(creatable).

The drawing.setModeCreate(circle.getCreatable()) function expects a helper shape called "creatable" that can be reached by the circle.getCreatable(), ellipse.getCreatable(), etc. functions.

Creatable helper shapes can have individual styles different from the shape that is finally created. To define the style for the helper shape, all functions of xrx.shape.Style such as line.getCreatable().setStyle() or polygon.getCreatable().setFillColor() are available. In the example below all shapes have transparent fill opacity during creation of shapes. The fill color disappears when the shape is finally created.
