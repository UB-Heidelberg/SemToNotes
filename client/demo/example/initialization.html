The size and the style of a drawing canvas can be defined with CSS or programmatically.

CSS:
CSS sizes can be absolute values such as #drawing1 { width: 200px; height: 200px; } or relative values, e.g., #drawing2 { width: 50%; min-height: 150px; }.
One can define all kinds of CSS styles for a drawing canvas such as #drawing4 { background-color: #D1F0FF; margin: 20px; padding: 20px; }.
Programmatically:
One can set the size programmatically with the drawing3.setSize(300, 200), drawing3.setWidth(300), drawing3.setHeight(200) functions that expect the width and height in pixel.
Styles can be also set programmatically, e.g., with drawing4.setStyle('padding', '20px').

Programmatic size and style definitions overwrite CSS rules.
