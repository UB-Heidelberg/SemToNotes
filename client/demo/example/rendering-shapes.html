Rendering Shapes

The following shapes are available: Circle, Ellipse, Line, Polygon, Polyline, Rectangle.

Create a new shape with the available shape constructors and by overloading the drawing canvas object, e.g., with new xrx.shape.Circle(drawing), new xrx.shape.Ellipse(drawing).

The geometry of a shape can be defined with functions such as circle.setCenter(100, 100) or polygon.setCoords([[200, 200], [320, 210], [310, 300], [305, 280]]). Coordinate points are expected as pixel values. See the API Reference for detailed function documentation of each shape.

In order to render shapes on the canvas, add them to the drawing canvas with drawing.addShapes(circle, ellipse).
