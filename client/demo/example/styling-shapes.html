Styling Shapes

The following styles are available: Fill Color, Fill Opacity, Stroke Color, Stroke Width.

One can define the style of a shape, e.g., with circle.setFillColor('rgb(153, 0, 153)') or with polygon.setFillOpacity(.2).

  • Stroke and fill colors can be defined as HTML colors such as green, rgb(153, 0, 153) or #805C6E.
  • Fill opacity expects a decimal number (.4 or 0.25) between 0 and 1, where 0 means full transparency and 1 means full color.
  • Stroke width expects an integer number such as 1, 2 or 5that represents the stroke width in pixel.
  • Stroke width by default is set to 1 pixel (black color). To make the stroke invisible you have to set the stroke width to 0. See the rectangle example below.
