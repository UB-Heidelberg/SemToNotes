Modifying Shapes

  • In order to modify shapes, switch a drawing canvas into mode modify, calling drawing.setModeModify(). Each Shape can then be moved as a whole by touching and dragging it.
  • When touching a shape, helper shapes appear (small circles) that can be used to change the geometry of a shape. For circle and ellipse shapes, the helper shape changes the radius / the radii. For all other shapes (line, polygon, poly-line, rectangle), the helper shape moves one coordinate point of the shape.
  • When modifying a shape, the drawing canvas recalculates all coordinate points and the sizes of a shape incrementally. See the console output on the right.
  • The example below also demonstrates the usage of the drawing.eventShapeModify(shape) event, which is thrown by a drawing canvas whenever a shape is modified.
  • The style of the helper shapes during modification can be defined individually, e.g., with circle.getModifiable().setStrokeColor() or ellipse.getModifiable().setStyle().
