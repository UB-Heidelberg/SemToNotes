Basic Template

  • Include the minified ./xrx.api.graphic.min.js source file into your HTML page, either in the HTML <head> or <body> element. Find the source at https://github.com/HKIKoeln/SemToNotes/blob/develop/client/xrx.api.graphic.min.js.
  • Prepare an HTML element such as <div id="drawing"><div> inside the HTML <body> element.
  • Write some JavaScript code that installs a drawing canvas at the <div id="drawing"><div> HTML element. Also create some shapes and insert them into the drawing canvas. See the various examples for more information.
  • Declare the size of your drawing canvas to make it visible, e.g., with CSS: #drawing { width: 100px; height: 100px; }.

The example below uses a simple self invoking JavaScript function inside the HTML page to install a drawing canvas. The graphic API can be used from within JavaScript libraries such as Angular or jQuery as well.
