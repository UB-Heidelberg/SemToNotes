Hovering Shapes

  • In order to style shapes differently when hovered, turn a drawing canvas into mode hover, e.g., with drawing1.setModeHover() or drawing2.setModeHover(true).
  • Sometimes shapes overlap. As a default, only the foremost shape is styled differently when hovered. One can change this default behaviour by setting the multiple option to true in the function call, e.g., with drawing2.setModeHover(true)
  • Each shape object automatically has a helper shape attached called hoverable that can be reached with the getHoverable() function. One can define the style of hovered shapes, e.g., with rect.getHoverable().setFillColor('#99CCFF') or circle.getHoverable().setFillColor('#99CCFF').
